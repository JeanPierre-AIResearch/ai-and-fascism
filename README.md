AI and Fascism — A Public Dialogic Research Project
This repository documents an ongoing public research project exploring the relationship between artificial intelligence and fascism.
Format
The research takes the form of a structured dialogue between Jean-Pierre (human researcher) and Claude (Anthropic AI). Both the questions and the answers constitute the research material. The dialogues are published in full — without editing or curation — in German and English.
Structure
dialogues/
  en/   → English dialogues
  de/   → German dialogues
Each dialogue is stored as a .md file and rendered as HTML via GitHub Pages.
Purpose
This is an open, transparent research process. The choice of a dialogic format reflects the conviction that the way we ask questions about AI and fascism is itself part of the research.
